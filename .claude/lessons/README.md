# Neovim Lessons

Quick reference guides for productive Neovim usage.

## Lessons

1. **[Feature Branch Workflow](01-feature-branch-workflow.md)** — Complete git workflow from branch creation to commit, using gitsigns, terminal, and Rubocop.

2. **[Text Objects and Motions](02-text-objects-and-motions.md)** — The grammar of Vim: verbs, nouns, and how to combine them for efficient editing.

3. **[Finding Things Fast](03-finding-things-fast.md)** — Telescope, grep, and search techniques for navigating any codebase.

4. **[Windows, Buffers, and Tabs](04-windows-buffers-and-tabs.md)** — How Neovim organizes files and views, and how to work with splits.

5. **[LSP Code Navigation](05-lsp-code-navigation.md)** — Using language servers for go-to-definition, find references, and refactoring.

## Quick Reference

| Keys | Purpose |
|------|---------|
| `<Space>sk` | Search keymaps (discover what's available) |
| `<Space>sh` | Search help documentation |
| `<Space>` (wait) | Which-key shows available options |
